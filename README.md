# Tic Tac Toe Game

## About
Tic Tac Toe is a classic two-player game where players take turns marking Xs and Os in a 3x3 grid. The player who places three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Features
- **Two-player mode**: Challenge a friend to a duel of wits.
- **Single-player mode**: Play against the computer.
- **Difficulty levels**: Choose between easy, medium, and hard modes when playing against the computer.
- **Responsive design**: Works seamlessly on various screen sizes.

## How to Run
1. Clone the repository: 
   ```
   git clone https://github.com/codedbyprashant/tic-tac-toe-game.git
   ```
2. Navigate to the project directory:
   ```
   cd tic-tac-toe-game
   ```
3. Open the `index.html` file in a web browser to start playing!

## Difficulty Levels
- **Easy**: The computer makes random moves.
- **Medium**: The computer makes strategic moves and can block winning moves.
- **Hard**: The computer plays an optimal game, making it challenging to win.

Have fun playing!
